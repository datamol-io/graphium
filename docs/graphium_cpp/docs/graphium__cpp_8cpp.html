◆ parse_mol()

std::unique_ptr< RDKit::RWMol > parse_mol ( const std::string & smiles_string,
bool explicit_H,
bool ordered = true )

Creates an RWMol from a SMILES string.

If ordered is true, and the string contains atom classes, called "bookmarks" in RDKit, that form a complete (0-based) ordering of the atoms, the atoms will be reordered according to this explicit order, and the bookmarks will be removed, so that canonical orders can be correctly compared later.

This is implemented in graphium_cpp.cpp, but is declared in this header so that both labels.cpp and features.cpp can call it.
