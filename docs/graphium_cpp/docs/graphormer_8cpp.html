std::vector< T > & all_pairs_distances )

Computes the shortest path distance, along edges, between all pairs of nodes, outputting to all_pairs_distances. Template type T can be float or double. Implementation is in graphormer.cpp

Parameters
nNumber of nodes
neighbor_startsArray of n+1 indices into neighbors, indicating where each node's neighbors start, plus one at the end to indicate the full length of neighbors
neighborsConcatenated array of all neighbors of all nodes, in order
queuevector used for temporary storage internally
all_pairs_distancesThis will be filled with the unweighted lengths of the shortest path between each pair of nodes.
